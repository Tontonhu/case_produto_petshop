# 🗺️ Roadmap do Produto: Pet Delivery App

Este documento descreve a visão de evolução do produto, priorizando entregas que geram valor imediato para o cliente e sustentabilidade para o negócio.

## 🚀 Fase 1: MVP & Retenção (Atual)
**Objetivo:** Estabilizar a base de clientes e reduzir o Churn logístico.

| Funcionalidade | Descrição Técnica | Impacto Esperado |
| :--- | :--- | :--- |
| **Rastreio Real-time** | Integração com API de Logística para disparos via WhatsApp. | Redução de 20% nas reclamações de atraso. |
| **Self-Service Portal** | Interface para o cliente alterar data de entrega sem suporte humano. | Redução de 15% nos chamados de SAC. |
| **Data Cleanup** | Saneamento da base de dados de endereços para evitar falhas de envio. | Aumento na taxa de sucesso de entrega na 1ª tentativa. |

## 📈 Fase 2: Engajamento & LTV (Próximos 3-6 meses)
**Objetivo:** Aumentar o tempo de permanência do cliente e o valor gasto.

| Funcionalidade | Descrição Técnica | Impacto Esperado |
| :--- | :--- | :--- |
| **Programa Pet VIP** | Motor de regras para aplicação de descontos progressivos (3º, 6º e 12º mês). | Aumento do LTV (Lifetime Value) em 10%. |
| **Indique e Ganhe** | Sistema de geração de cupons únicos para indicação de novos membros. | Redução do CAC (Custo de Aquisição de Cliente). |
| **Dashboard PO 2.0** | Migração dos relatórios de Excel para uma visualização em Power BI/Looker. | Tomada de decisão 2x mais rápida. |

## 🔮 Fase 3: Expansão & Inteligência (Futuro)
**Objetivo:** Escalar o negócio e personalizar a experiência.

| Funcionalidade | Descrição Técnica | Impacto Esperado |
| :--- | :--- | :--- |
| **Predição de Consumo** | Algoritmo que sugere o timing de compra baseado no peso do pet. | Prevenção de churn por falta de estoque. |
| **Cross-selling** | Recomendação de itens complementares (petiscos/brinquedos) no checkout. | Aumento de 15% no Ticket Médio. |
